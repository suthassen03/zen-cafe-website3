ZEN CAFE – FERDIG VERCEL-PROSJEKT

1. Pakk ut ZIP-filen.
2. Åpne mappen. Du skal se:
   - index.html
   - images-mappen
3. Gå til vercel.com og logg inn.
4. Velg Add New → Project.
5. Velg "Deploy without Git" eller last opp mappen.
6. Dra hele mappen "zen-cafe-complete" inn i opplastingsfeltet.
7. Trykk Deploy.
8. Når publiseringen er ferdig, får du en Vercel-lenke.

VIKTIG:
Ikke last opp bare index.html. Last opp hele mappen, slik at images-mappen følger med.

ENDRE TEKST:
Åpne index.html i TextEdit eller Visual Studio Code.
Søk etter navn eller pris og endre teksten.

ENDRE HASTIGHET:
Søk etter:
animation: rotateCarousel 24s linear infinite;

15s = raskere
24s = anbefalt
35s = saktere
